# Pangea StratBal++ (aka "Fish Map Script")

A Civilization V map script: Pangaea-style continent generation that
guarantees every player gets Coal and Aluminum in their strategic balance,
instead of leaving those two resources to chance.

Created by **Fish** (credited in the `.modinfo` as author). This repo just
mirrors the file exactly as it's already known and distributed in the
Democraciv community -- folder name, file names, and the `.modinfo`'s
internal `Name`/`EntryPoint` text are all untouched, byte-for-byte
identical to what players already have installed. "Pangea StratBal++" is
only this *repo's* label; nothing inside the shipped files was renamed, so
existing installs aren't treated as a different/newer thing and don't need
to be replaced.

Pulled automatically by [kek-mod's installer](https://github.com/OBLASTWAR/kek-mod)
if a player doesn't already have it.

## Manual install

Drop the contents of this repo into:

```
Sid Meier's Civilization V/Assets/Maps/Fish Map Script/
```

It'll show up as "HellBlazers Map Script" in the map script dropdown at
game setup (that's the original entry point name) -- no need to enable it
via the Mods menu.
