# Pangea StratBal++

A Civilization V map script: Pangaea-style continent generation that
guarantees every player gets Coal and Aluminum in their strategic balance,
instead of leaving those two resources to chance.

Originally created by **Fish** (credited in the mod's `.modinfo` as author).
This repo repackages and redistributes it for the Democraciv community,
pulled automatically by [kek-mod's installer](https://github.com/OBLASTWAR/kek-mod)
if it isn't already present.

## Manual install

Drop the contents of this repo into:

```
Sid Meier's Civilization V/Assets/Maps/Pangea StratBal++/
```

It'll show up as "Pangea StratBal++" in the map script dropdown at game
setup -- no need to enable it via the Mods menu.
